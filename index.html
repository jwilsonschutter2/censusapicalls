Get a Mapbox token

To use Mapbox maps, you need a public access token (starts with pk). [1](https://docs.mapbox.com/help/getting-started/access-tokens/)[2](https://docs.mapbox.com/help/glossary/access-token/)

  1. Sign up / log in at mapbox.com
  2. Go to your Access Tokens page in the Mapbox console
  3. Copy your Default public token (or create a new public token) [1](https://docs.mapbox.com/help/getting-started/access-tokens/)[4](https://docs.mapbox.com/accounts/guides/tokens/)
  4. Paste it into the token box and click Save token

Tip: For public websites, Mapbox supports URL restrictions/scopes to limit token use. [1](https://docs.mapbox.com/help/getting-started/access-tokens/)[4](https://docs.mapbox.com/accounts/guides/tokens/)
